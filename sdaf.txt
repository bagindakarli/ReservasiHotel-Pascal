Bagian ini berisi gambaran umum aplikasi/sistem yang akan dibuat. Juga berisi ruang lingkup dan batasan sistem.
Contoh : Aplikasi ini bertujuan untuk mempermudah pencatatan proses penyewaan mobil. Aplikasi ini akan mencakup proses penyewaan, pengembalian, pembayaran, dan juga denda. Aplikasi focus hanya pada proses penyewaan sehingga tidak mencakup kehilangan dan kecelakaan yang terjadi selama penyewaan.

B. DATA
Aplikasi ini menggunakan 3 buah Tipe Data bentukan, yaitu :
Mobil : <Nopol, Kapasitas, Harga>, untuk menyimpan data mobil
Penyewa : <NoKTP, Nama, Alamat>, untuk menyimpan data penyewa
Penyewaan : <Tanggal, NoKTPPenyewa, NopolMobil>, untuk menyimpan data penyewaan mobil

dan 3 buah Array, yaitu :
ArrayMobil : Mobil(Tipe dari array), untuk menyimpan data semua mobil yang dimiliki perusahaan
